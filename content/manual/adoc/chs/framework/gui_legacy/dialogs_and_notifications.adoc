[[dialogs_and_notifications]]
==== 对话框消息和通知消息（历史版本）

[WARNING]
====
这是旧版本 API。从 7.0 版本开始的新 API，请参考 <<gui_dialogs>> 和 <<gui_notifications>>。
====

对话框消息和通知消息可以用来为用户呈现消息。

对话框消息有标题、关闭按钮并且总是在应用程序主窗口的中间展示。通知消息可以显示在窗口中间或者角落，并且能自动消失。
